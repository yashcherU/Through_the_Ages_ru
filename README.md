# Улучшенная русификация игры Through the Ages_gog

<strong>Особенности:</strong>

Улучшено качество перевода:

- исправлены опечатки, неточности перевода, восполнены ранее отсутствующие фрагменты текста путём сравнения файлов на других языках;
- текстовое наполнение карточек переработано для максимальной точности смысла;
- терминология приведена к единому стилю;
- текст в журнале игроков стал более живым и выразительным.

Улучшено визуальное оформление:

- читаемость заголовков карточек на маленьких экранах улучшилась за счёт использования прописных символов;
- читаемость текста на карточках улучшилась за счёт масштабирования размера шрифта после переработки излишне объёмного текста;
- по возможности длина текста подгонялась так, чтобы он ложился ровнее и аккуратнее на карточку;
- стандартные иконки в тексте карточек используются чаще для лучшего восприятия.

<p> </p>
<strong>Порядок установки:</strong>

- если игра устанавливается впервые, запустить её и загрузить существующую локализацию игры.
- выйти из игры и поместить с заменой содержимое архива Through_the_Ages_ru.zip в папку с игрой (c:\GOG Games\Through the Ages\data_ram\Localization_strings\)
